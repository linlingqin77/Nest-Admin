# 增值税及附加税申报需求文档

## 知识点
### 增值税基本知识
> 增值税是对商品生产、流通、劳务服务中多个环节的新增价值或商品的附加值征收的一种流转税。

### 纳税人类型
| 类型 | 说明 | 税率 |
| --- | --- | --- |
| 一般纳税人 | 年应税销售额超过500万元的企业 | 13%、9%、6%、0% |
| 小规模纳税人 | 年应税销售额不超过500万元的企业 | 3%（减按1%）、5% |

## 小规模纳税人免税政策说明

| 政策 | 说明 |
| --- | --- |
| 月度免税额度 | 月销售额不超过10万元（含10万元）免征增值税 |
| 季度免税额度 | 季度销售额不超过30万元（含30万元）免征增值税 |
| 减按1%征收 | 2023年1月1日至2027年12月31日，增值税小规模纳税人适用3%征收率的应税销售收入，减按1%征收率征收增值税 |
| 六税两费减半 | 小规模纳税人可享受城市维护建设税、教育费附加、地方教育附加等"六税两费"减半征收优惠 |
